# MBA_USP_DATA_SCIENCE_2026

Resumo
Este trabalho investiga o uso de um pipeline de engenharia de features não supervisionada — denominado UnsupervisedFeaturizer — para aprimorar classificadores supervisionados aplicados a problemas de detecção de anomalias com severo desbalanceamento de classes. O pipeline combina quatro técnicas complementares: K-Means (agrupamento), PCA (redução de dimensionalidade), GMM (modelagem de densidade probabilística) e LOF (detecção de outliers local), cujas representações são concatenadas às features originais antes do treinamento dos classificadores.
Foram avaliados dois classificadores — Regressão Logística (RL) e Random Forest (RF) — em quatro cenários experimentais (C1–C4), comparando o pipeline proposto com abordagens de referência amplamente utilizadas: SMOTE (sobreamostragem sintética) e ponderação de custo (cost-sensitive learning). Os experimentos foram conduzidos em dois conjuntos de dados: um dataset sintético controlado e o dataset de mamografia (Mammography), com repetições N=30 e teste estatístico t pareado para validação das diferenças observadas.
As métricas de avaliação incluíram PR-AUC, ROC-AUC, F1-Score, Precisão, Recall, Índice de Fisher J1 e Brier Score. Os resultados demonstram que o UnsupervisedFeaturizer, especialmente no cenário C4 (features originais + não supervisionadas + interações), produz ganhos consistentes sobre as abordagens de referência, com desempenho de RL comparável ao melhor resultado do RF em determinadas métricas.
